Cada semana tiene un objetivo claro, contenidos definidos y un mini-proyecto que consolida lo aprendido.

01
Semana 1 — Fundamentos sólidos
Consolidación de Java y POO
Empezamos diagnosticando tu nivel real y cubriendo todos los fundamentos que todo desarrollador Java debe dominar. Orientación a objetos aplicada a código real, no a ejemplos de libro. Al final de la semana tendrás una mini app de consola funcional.
Tipos y variables Control de flujo Arrays y Strings Clases y objetos Herencia Polimorfismo Interfaces Mini-proyecto: gestor de tareas CLI
02
Semana 2 — Java moderno
Funcional, Streams y Colecciones
Aquí es donde muchos estudiantes se quedan atascados. Veremos lambdas y Streams de forma progresiva, con casos de uso reales de negocio que hacen que el concepto encaje de forma natural. Dominarás el Java que se usa hoy en la industria.
List · Set · Map Generics Lambdas Stream API Interfaces funcionales Optional Method references Mini-proyecto: procesado de datos con Streams
03
Semana 3 — Nivel avanzado
Patrones de diseño, SOLID y Control de Versiones
El salto cualitativo más importante. Aprenderás a pensar en diseño de software y flujos con Git, no solo en código que funciona. Refactorizaremos código real aplicando los patrones y principios que distinguen a un junior de un mid/senior.
SOLID (SRP · OCP · LSP · ISP · DIP) Factory Strategy Observer Builder Decorator Singleton Clean Code · Naming · Refactoring
04
Semana 4 — Proyecto final
Testing, arquitectura limpia y entrega
La semana más satisfactoria del programa. Construirás el proyecto final desde cero con todo lo aprendido: arquitectura en capas, TDD, tests con JUnit 5 y Mockito, y una revisión de código profesional. El resultado es algo que puedes enseñar en una entrevista o subir a GitHub con orgullo.
TDD JUnit 5 Mockito AssertJ Arquitectura en capas Code review profesional Proyecto entregable Certificado

Dudas sobre el intensivo

¿Qué nivel necesito para apuntarme al intensivo?
+
El plan está pensado para quienes tienen nociones básicas de programación (en cualquier lenguaje) o de Java. Si partes de cero absoluto, te recomiendo empezar con 2–3 sesiones particulares antes del intensivo para que el ritmo no se haga cuesta arriba. Lo evaluamos juntos en la sesión de diagnóstico gratuita.
¿Cuándo empiezan las clases?
+
Las plazas para julio y agosto se cubren rápido. El horario lo pactamos tú y yo según tu disponibilidad. Normalmente las sesiones son de mañana o tarde, de lunes a viernes, aunque podemos ajustarlo.
¿El precio es por persona o por grupo?
+
Los 199€ son por persona. Si venís dos amigos o compañeros juntos, podemos negociar un precio especial de grupo. Escríbeme y lo hablamos.
¿Qué pasa si me pierdo una sesión?
+
Las sesiones son recuperables. Si avisas con al menos 24h de antelación, reagendamos sin problema. Las clases pueden grabarse con tu permiso para repasar contenido.
¿Cuánto tiempo al día necesito dedicar fuera de clase?
+
Entre 1 y 2 horas diarias de práctica para consolidar lo visto en sesión. No es obligatorio, pero el aprendizaje mejora exponencialmente si practicas a diario. Te doy ejercicios guiados para que no tengas que buscar qué hacer.
